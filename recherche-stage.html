Application de recherche de stage

Reprise d'une application déjà existente de recherche de stage, où nous devions l'améloirer et y ajouter de nouvelle fonctionalités, telle que l'import de stage par fichier. Pour ce projet nous étions en équipe de 5, pour une durée d'une semaine. Le projet comporter un site web en php utilisant le framwork symfonie, et d'une API elle aussi faite en php. D'une application android qui utilisé l'API pour enregistrer les données.
